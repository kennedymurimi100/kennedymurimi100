## Ordered List of Steps on Making a Commit in Git

1. Check the status – Run `git status` to see which files have been modified, added, or deleted.
2. Write a commit message – Prepare a clear, descriptive message explaining the changes following project conventions.
3. Verify the commit – Use `git log` to confirm your commit was created successfully with the correct message and files.
